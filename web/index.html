02 · The split

OpenClaw and Hermes give you a runtime — a place for agents to live.
Harbor gives you a control plane — connect, plan, run, persist.

You don't need another chat window. You need a harbor — somewhere your builder loop docks, loads context, and ships.

Harbor orchestrates five sponsors into one proprietary loop. OpenClaw can host the runtime; Harbor owns the memory, integrations, workspace state, and multi-step workflows.

03 · Why Harbor is different

Not a chatbot. Not a cron job. Not a demo.

What already exists vs what Harbor adds.

OpenClaw / Hermes

  • Agent runtime & messaging gateway
  • You wire tools and prompts yourself
  • Memory is your problem
  • No project workspace or run history
  • Great for hosting — not for builder ops

Harbor

  • Pre-wired 5-sponsor agent loop
  • SuperCompress before every inference
  • Projects, plans, briefs in .harbor/
  • Integration picker + OAuth (solo-friendly)
  • Run history with KV savings & turn logs

Random scripts / Zapier

  • One integration per script
  • No research or reasoning layer
  • No memory compression
  • Fragile, not agentic
  • Doesn't plan — just triggers

05 · Why each sponsor matters

Five tools. One loop. None optional.

Harbor isn't "integrated with sponsors" — it's built from them. Each one solves a layer generic agents skip.

Gather Tavily

Live web intel before every run — competitor moves, outage status, docs you haven't read yet. Chatbots guess; Harbor searches.

Act Composio

OAuth to your real GitHub, Linear, Gmail. Not mock tools — actual PRs, tickets, and inbox threads in the agent loop.

Remember SuperCompress

Proprietary memory layer. Trained on your workflow context. Without it, multi-step agent runs hit token walls and forget.

Think Nebius Token Factory

Multi-turn tool-calling inference — the reasoning engine that decides what to do with gathered context.

Host (optional) OpenClaw

24/7 runtime bridge via SKILL.md — not the product. Harbor works standalone via CLI + local dashboard.

Persist Harbor workspace

.harbor/ — projects, plans, briefs, run history, KV stats. The thing no runtime gives you for free.

07 · Install

Running in 60 seconds

$ git clone https://github.com/arjunkshah/harbor.git
$ cd harbor && ./bootstrap.sh
$ source .venv/bin/activate && harbor setup
$ harbor serve
→ http://127.0.0.1:8787/dashboard

No keys? harbor demo. Then harbor run "plan my launch" --plan or harbor brief.
