Documentation

Install Harbor, connect your apps, run your first brief.

This site is the project homepage (docs + install). The dashboard (integrations, run history, live briefs) runs locally via harbor serve — it is not hosted on GitHub Pages.

Setup

git clone https://github.com/arjunkshah/harbor.git
cd harbor
./bootstrap.sh
source .venv/bin/activate
harbor setup

harbor setup writes .env, trains SuperCompress, walks you through Composio OAuth, and runs harbor doctor.

.env (minimal)

NEBIUS_API_KEY=
COMPOSIO_API_KEY=
TAVILY_API_KEY=
COMPOSIO_TOOLKITS=github,linear,gmail
HARBOR_USER_ID=harbor-builder-001
HARBOR_DEMO=0

All workflow targets (GitHub owner/repo, Linear team, Slack channel) are optional.

CLI commands

  • harbor setup — interactive wizard
  • harbor doctor — verify stack
  • harbor brief — morning brief → terminal + .harbor/briefs/latest.md
  • harbor incident "…" — incident commander
  • harbor connect github — Composio OAuth
  • harbor demo — no API keys
  • harbor serve — local web + dashboard + API

Local dashboard

The dashboard is a local-only UI for integration management and run history:

harbor serve
# open http://127.0.0.1:8787/dashboard

Or: harbor dashboard (starts the server and opens the browser).

  • Integration health & masked key status
  • Trigger briefs and incidents from the browser
  • Browse run history, KV savings, turn logs

Integrations (solo-friendly)

GitHub

Connect once via OAuth — Harbor searches your whole account (PRs, issues, commits). No repo config required.

Linear

Regular Linear workspace. Auto-picks your default team when creating tickets.

Gmail

Personal inbox triage — unread and important threads.

Slack (optional)

Only if you broadcast to a team channel. Add slack to COMPOSIO_TOOLKITS. Solo builders can skip entirely.

OAuth connect

harbor connect github
harbor connect linear
harbor connect gmail
