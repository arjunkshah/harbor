Quick start

Running in 60 seconds

install
$ git clone https://github.com/arjunkshah/harbor.git
$ cd harbor && ./scripts/install_and_demo.sh
$ cp .env.example .env # add BuilderShip sponsor keys
$ harbor doctor && harbor brief
