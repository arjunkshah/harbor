Documentation

Install Harbor, connect your apps, run your first agent task.

This site is the project homepage (docs + install). The builder workspace — Connect, Projects, Agent, Plans, History — runs locally via harbor serve. It is not hosted on GitHub Pages.

Setup

git clone https://github.com/arjunkshah/harbor.git
cd harbor
./bootstrap.sh
source .venv/bin/activate
harbor setup

harbor setup writes .env, trains SuperCompress, lets you pick integrations, walks Composio OAuth, and runs harbor doctor.

API keys

Register at ship.builders for BuilderShip sponsor credits:

  • Nebius — tokenfactory.nebius.com
  • Composio — dashboard.composio.dev
  • Tavily — app.tavily.com

.env (minimal)

NEBIUS_API_KEY=
COMPOSIO_API_KEY=
TAVILY_API_KEY=
COMPOSIO_TOOLKITS=github,linear,gmail
HARBOR_USER_ID=harbor-builder-001
HARBOR_DEMO=0

CLI commands

  • harbor setup — interactive wizard
  • harbor doctor — verify stack
  • harbor run "…" — general builder agent task
  • harbor run "…" --plan — plan only → saves to .harbor/plans.json
  • harbor brief — morning brief
  • harbor incident "…" — incident commander
  • harbor connect github — Composio OAuth
  • harbor integrations list|set — choose toolkits
  • harbor demo — no API keys
  • harbor serve — local web + workspace dashboard
  • harbor dashboard — serve + open browser

Local workspace

harbor serve
# open http://127.0.0.1:8787/dashboard

The dashboard is a sidebar workspace with views:

  • Overview — stats, differentiators, stack health
  • Board — kanban for plans, PRD features, and coding jobs
  • Connect — toggle integrations, OAuth, Gmail sync settings
  • Projects — what you're building
  • Agent — run tasks, briefs, incidents
  • Plans — agent roadmaps with checkable tasks
  • History — run log with KV savings and turn traces

Integrations (solo-friendly)

Harbor ships a catalog of 12 Composio toolkits. Defaults: GitHub + Gmail. Linear is optional — use Harbor Board for planning instead.

GitHub

OAuth to your whole account — PRs, issues, commits. No repo config required.

Gmail

Sync emails auto-send by default when plans/PRD sync. Switch to drafts in Connect → Settings or HARBOR_GMAIL_SYNC_MODE=draft.

Also available

Slack, Notion, Google Calendar, Sheets, Drive, Discord, Trello, Jira, Airtable, Asana, Linear (optional).

harbor integrations set github,gmail,notion,slack
harbor connect github
harbor connect gmail

Why not OpenClaw alone

Harbor is a builder control plane, not a prompt pasted into OpenClaw:

  • Multi-sponsor orchestration loop (Tavily → Composio → SuperCompress → Nebius)
  • Persistent .harbor/ state: projects, plans, briefs, run history
  • Integration catalog with solo-friendly defaults
  • Plans and general harbor run tasks — not just chat

OpenClaw remains an optional runtime bridge via /openclaw/chat and SKILL.md.
