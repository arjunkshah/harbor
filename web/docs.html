Documentation

Everything you need to install Harbor, connect sponsor APIs, and ship.

Setup (recommended)

Run the interactive wizard — it writes your .env, trains memory, connects Composio, and runs doctor.

git clone https://github.com/arjunkshah/harbor.git
cd harbor
pip install -e .
harbor setup
Or use the install script: ./scripts/install_and_demo.sh then harbor setup for live keys.

CLI commands

  • harbor setup — interactive API key + OAuth wizard
  • harbor doctor — verify all integrations
  • harbor brief — morning brief workflow
  • harbor incident "…" — incident commander
  • harbor serve — web + dashboard + API
  • harbor dashboard — open dashboard (starts server)
  • harbor connect github — Composio OAuth link
  • harbor demo — no keys required

Dashboard

The dashboard requires the local server (workflows hit your machine's API keys):

harbor serve
# → http://localhost:8787/dashboard

From the dashboard you can:

  • See integration health & masked key status
  • Trigger morning briefs and incident runs
  • Browse run history, KV savings, and turn logs

Stack integration

Tavily

Search, multi-search, URL extract, Q&A synthesis, company intel, social pulse — all used in agent gather phase.

Composio

GitHub, Slack, Linear, Gmail — read during gather, write during act (Slack digest, Linear tickets).

Nebius Token Factory

OpenAI-compatible chat completions with tool calling every agent turn.

SuperCompress

Compresses Tavily + Composio context before each Nebius call (~35% token budget).

OpenClaw

Webhook at /openclaw/chat, skill in openclaw/SKILL.md, cron example included.
