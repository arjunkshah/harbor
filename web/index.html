What it is

A builder-ops agent in your terminal

Harbor is an open-source CLI that runs autonomous workflows across the BuilderShip sponsor stack. Connect your GitHub account once, add API keys, run harbor brief, and get a synthesized digest of what needs your attention — plus optional follow-up actions.

How it works

Gather → compress → think → act

Every workflow runs the same agent loop — not a single API call.

Install

Running in 60 seconds

install
$ git clone https://github.com/arjunkshah/harbor.git
$ cd harbor && ./bootstrap.sh
$ source .venv/bin/activate
$ harbor setup
$ harbor brief

No keys yet? harbor demo runs with fixtures. Full setup docs →
